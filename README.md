# 🌍 EcoClick  

EcoClick es una aplicación educativa gamificada orientada a niños de educación básica.  
Su objetivo es enseñar sobre **reciclaje, biodiversidad y cuidado del entorno** a través de mini-juegos interactivos, con sistema de puntajes, logros y retroalimentación visual.  

El proyecto está desarrollado con:  
- **Frontend:** Dart con **Flutter Web**  
- **Backend:** JavaScript con **Node.js y Express**  

---

## 🚀 Demo  

El proyecto incluye una **demo funcional** con una pantalla principal, mini-juegos y sistema de puntajes.  

### 1. Requisitos previos  

Antes de ejecutar el proyecto asegúrate de tener instalado:  

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (3.0 o superior)  
- [Node.js](https://nodejs.org/) (versión 16 o superior)  
- Navegador moderno (Chrome, Edge, Firefox).  

Opcional:  
- [Postman](https://www.postman.com/) para probar las rutas del backend.  

---

### 2. Clonar el repositorio  

### 3. Backend (JavaScript con Node.js)

1. Accede a la carpeta del backend:
   
```bash
cd backend
```
2. Instala dependencias:
```bash
npm install
```
3. Inicia el servidor:
```bash
npm start
```
o
```bash
npm run dev
```
Por defecto corre en http://localhost:3000/.

### 4. Frontend (Dart con Flutter Web)

1. Accede a la carpeta del frontend:
   
```bash
cd frontend
```

2. Instala dependencias

```bash
flutter pub get
```

3. Ejecuta el servidor de desarrollo:
   
```bash
flutter run -d chrome (o navegador de tu preferencia)
```
Esto abrirá la aplicación en tu navegador (http://localhost:8080/).
